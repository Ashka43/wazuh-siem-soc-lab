# Wazuh SIEM SOC Lab

## Overview
This project demonstrates the deployment of a SIEM solution using Wazuh to monitor a Linux endpoint, collect security logs, and detect suspicious activities through centralized analysis.

## Lab Architecture
- Wazuh Manager & Dashboard: Ubuntu Server
- Endpoint Agent: Kali Linux
- Network: Virtualized environment

## Tools Used
- Wazuh SIEM
- Kali Linux
- Ubuntu Server
- Linux logging and monitoring tools

## Key Features
- Secure onboarding of Linux agent
- Centralized log collection
- Detection of authentication failures
- Security event visualization in dashboard

## Security Use Case Demonstrated
- Detection of failed authentication attempts
- Log analysis and alert investigation

## Screenshots
Screenshots demonstrating agent connectivity, security alerts, and log evidence are included.

## Learning Outcomes
- Understanding SIEM architecture
- Agent-manager communication
- Log-based threat detection
- SOC-style alert analysis
